# mixplus-ctl

mix-plus-cli init mix-plus go skeleton


# Install
```go
go install github.com/mix-plus/go-mixplus/tools/mpctl@latest
```

# Run
```
// init grpc
mpcli new mrpc
// init api 
mpcli new api
// init queue
mpcli new queue
```
